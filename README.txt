AppTravel trata-se de um projeto de TCC onde utlizo as tecnologias angular 8 para frontend, .net c# para api backend e banco de dados SQL Server. 

O sistema foi elaborado para realizar recomendações de locais turísticos, baseando-se nas avaliações dos locais pelos usuários. 

Ao baixar o projeto, deverá executar o comando npm install no cmd NodeJs para carregar todos os pacotes necessários para rodar o projeto em Angular.
A pasta DB_BKP contem o backup do banco de dados utilizado na apresentação.
